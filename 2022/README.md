# Advent of Code 2022
C++ solutions for AOC 2022 event

## Build and Run
you can build solutions for each day using any c++ compiler version >= 14. And then just pipe your input to the input buffer.

***Example:***
```bash
cd d12
g++ -o ./a.exe ./a.cpp
./a.exe < input.txt
```
